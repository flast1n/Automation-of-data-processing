# АРМ Оператора АТ «Укрпошта»

Програмний комплекс для автоматизації робочого місця оператора зв'язку та адміністратора відділення «Укрпошта». Додаток забезпечує швидкий облік відправлень, роботу з базою клієнтів та аналітику для керівництва.

## ✨ Основний функціонал

* **Розмежування прав доступу:** Реалізовано дві ролі користувачів (`admin` та `operator`) з безпечною авторизацією та шифруванням паролів за алгоритмом SHA-256.
* **Модуль Оператора:**
  * Пошук та трекінг посилок за унікальним 13-значним номером.
  * Швидка реєстрація нових клієнтів.
  * Оформлення відправлень (розрахунок ваги, вартості, вибір типу доставки).
* **Модуль Адміністратора:**
  * Перегляд фінансових звітів відділення (каса).
  * Аналітика ефективності роботи операторів.
  * Журнал системних логів (`ActionLog`) для повного аудиту дій персоналу.
* **Безпека даних:** Суворий контроль реляційної цілісності на рівні СКБД (`PRAGMA foreign_keys = ON`) та інтерактивна валідація введених даних (захист від помилок).

## 🛠 Стек технологій

* **Мова програмування:** Python 3.11+
* **Графічний інтерфейс (GUI):** CustomTkinter (сучасна обгортка для Tkinter з підтримкою тем та кастомних віджетів).
* **База даних:** SQLite3.

## 📂 Структура проєкту

* `main.py` — точка входу, логіка графічного інтерфейсу та взаємодія віджетів.
* `database.py` — шар роботи з базою даних (усі SQL-запити, ініціалізація таблиць та хешування).
* `ukrposhta_db.db` — файл локальної реляційної бази даних SQLite.
